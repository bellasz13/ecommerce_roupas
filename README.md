# Trend Store - E-commerce de Roupas

## Descrição

O Trend Store é uma plataforma de e-commerce dedicada à venda de roupas, oferecendo uma experiência de compra moderna, intuitiva e responsiva para os usuários. O sistema permite gerenciamento de produtos, categorias, usuários, carrinho de compras, pedidos e pagamentos.

## Funcionalidade

* Catálogo de produtos de roupas femininas, masculinas e acessórios
* Cadastro e login de usuários
* Carrinho de compras com controle de quantidade e preços
* Processo de checkout com opções de pagamento

## Tecnologias Utilizadas

* Backend: Node.js 
* Banco de dados: My SQL
* Frontend: HTML, CSS, JavaScript 
* Controle de versão: Git + GitHub.
